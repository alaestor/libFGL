Detailed Description

template<std::contiguous_iterator T_iter, std::size_t T_dimensions>
class fgl::dynamic_linear_matrix_alias< T_iter, T_dimensions >

A linear matrix alias which prioritizes run-time flexibility.

Unlike the static_linear_matrix_alias, the bounds and major order of the dynamic alias can be changed at run-time. However the number of dimensions must still be known at compile-time.

Template Parameters
T_iterThe underlying linear contiguous iterator type
T_dimensionsThe number of dimensions
