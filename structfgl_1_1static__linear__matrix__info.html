Detailed Description

template<major T_major, std::size_t ... T_bounds>
struct fgl::static_linear_matrix_info< T_major, T_bounds >

A static container for linear matrix alias information, such as dimensions, bounds, and offsets.

Remarks
This class is used by internally by static_linear_matrix_alias. Using the static alias doesn't require interacting with this type directly. However, It remains part of the public interface as it may be useful for implementing other Linear Matrix Alias (utilities) abstractions.
Template Parameters
T_majorThe major order of the matrix
T_boundsThe multi-dimensional bounds of the matrix
