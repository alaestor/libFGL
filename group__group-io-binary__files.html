Reads a binary file into a contiguous range of bytes.

Parameters
file_pathThe path to the file to read
[out]output_rangeA contiuogus range of bytes to write the file contents to. It's size must be greater than or equal to bytes_to_read, or the size of the file if bytes_to_read is 0. For concept details, refer to Range Constraints.
bytes_to_readThe number of bytes to read from the file. If this is 0, the size of the file will be used. This must be less than or equal to the maximum representable value of std::streamsize.
Note
it's asserted that bytes_to_read, or the size of the file if bytes_to_read is 0, is less than or equal to the size of the maximum representable value of the type of std::streamsize.
Returns
The number of bytes read from the file. This will always be bytes_to_read unless bytes_to_read is 0, in which case it will be the size of the file.
Exceptions
std::filesystem::filesystem_errorIf bytes_to_read was 0 and the file size couldn't be obtained.
std::runtime_errorif the buffer is less than bytes_to_read, or the file couldn't be opened via std::ifstream
[various]Standard std::ifstream exceptions which are enabled for badbit, failbit, and eofbit.

Writes a number of bytes from a contiguous range of bytes to a binary file.

Parameters
file_pathThe path to the file to write
inputA contiguous range of bytes to write to the file For concept details, refer to Range Constraints.
bytes_to_writeThe number of bytes to write to the file. Must be less than or equal to the size of the input range, and less than the maximum representable value of std::streamsize.
modeThe mode to open the file with. Defaults to std::ios::trunc.
Note
it's asserted that bytes_to_write is less than the maximum representable value of the type of std::streamsize.
Exceptions
std::runtime_errorif the file couldn't be opened via std::ofstream
std::invalid_argumentif bytes_to_write is greater than the size of the input range
[various]Standard std::ofstream exceptions which are enabled for badbit and failbit.
