Detailed Description

template<fgl::traits::steady_clock T_clock>
class fgl::debug::output_config< generic_stopwatch< T_clock > >

A template for a template specializion of output handler for using stopwatches with libFGL's Debug Output System.

Note
The channel state is specific to each synthesizations of the stopwatch-specialized output_config template. To disable output from all synthesized stopwatch channels, set fgl::debug::disable_stopwatch_output_channels to true
See also
Debug Output System and fgl::debug::output::operator()()
Template Parameters
T_clockThe clock type to use for the stopwatch.
