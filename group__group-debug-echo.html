Detailed Description

Send messages or code expressions with their evaluated result to libFGL's Debug Output System.

A simple way to output a message, or an expression with its result, to the Debug Output System. Echo can also be used as a nice replacement for quick and dirty "print debugging".

See also
the example program example/fgl/debug/echo.cpp

#define FGL_DEBUG_ECHOV (   expression)

Send an expression with source location info and it's evaluated result to libFGL's Debug Output System on the "echo" channel.

Note
the expression is evaluated as an argument in a function call
Parameters
expressionThe expression which will be stringified and evaluated. The result of the expression will be echoed.
Note
If NDEBUG is defined, this macro expands to the in-place expression which will always be evaluated. No output will be sent.
