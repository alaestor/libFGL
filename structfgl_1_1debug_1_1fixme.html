Detailed Description

Used to specialize fgl::debug::output_config<fixme> is sent to the fgl::debug::output() when a fix-me message is emitted.

Remarks
In the vast majority of cases, this object type will only be used in the implementations of the FIX(), FIX_ME(), and FIX_THIS macros. You probably don't want to use this directly, but could if so desired.
