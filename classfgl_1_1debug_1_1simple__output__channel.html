Detailed Description

template<bool T_enabled, priority T_priority, fgl::string_literal T_name, typename T_key = void>
class fgl::debug::simple_output_channel< T_enabled, T_priority, T_name, T_key >

Configurable output_channel implementation, provided for convenience.

An implementation of an output channel which provides public configuration "setters" for priority and channel name. As this is a very common implementation, it's provided here as an extensible type which is intended to be inherited.

Template Parameters
T_enabledthe initial enabled state of the channel
T_prioritythe initial priority of the channel
T_namethe initial name of the channel
T_keya type used to differentiate between otherwise identical channel synthesizations.
Note
to ensure that the correct synthesization is being referred to across compilation units, the T_name string literal should be defined as static inline constexpr fgl::string_literal
