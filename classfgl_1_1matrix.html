Detailed Description

template<typename T, std::size_t ... T_bounds>
class fgl::matrix< T, T_bounds >

A multi-dimensional, row-major array.

Combines the behavior of a static_linear_matrix_alias with a std::array .

See also
the example program example/fgl/utility/matrix.cpp
Note
The range and its iterators are linear; multi-dimensional indexing is only possible thru the container.
Template Parameters
TThe matrix element type.
T_dimensionsThe number of dimensions
Examples
example/fgl/utility/matrix.cpp.
