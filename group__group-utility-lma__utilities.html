inlineconstexprnoexcept

Asserts that the ranges are of equal size, and then returns that size.

If T_dimensions is 0 , asserts that the sizes of a and b are the same and returns the size. Otherwise, asserts the sizes of a and b are equal to T_dimensions , and then returns T_dimensions .

Template Parameters
T_dimensionsThe compile-time number of dimensions. If this is 0 , only the sizes of a and b are compared. Otherwise, the sizes of a and b are compared to T_dimensions .
Returns
The size of a and b .

inlineconstexprnoexcept

Calculates and assigns dimensional offsets for the T_major of bounds .

Template Parameters
T_dimensionsThe compile-time number of dimensions. Must be either equal the size of both ranges, or 0 in which case the number of dimensions will be dynamically determined by the sizes of the ranges.
T_majorThe major extent to calculate offsets for.
T_boundsThe integral range type of the multi-dimensional bounds.
T_offsetsThe output range type which is assignable from the underlying type of T_bounds .
Parameters
boundsThe multi-dimensional bounds of the aliased range.
[out]out_offsetsA range to which the calculated offsets will be assigned. The underlying type of the range must be assignable from the underlying type of the bounds range. In other words, if the bounds type is T, the offsets type must be assignable from T . For concept details, refer to Range Constraints.
Examples
example/fgl/utility/linear_matrix_alias/utilities.cpp.
