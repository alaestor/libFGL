Detailed Description

template<fgl::traits::pointer_to_byte T>
struct fgl::cbyte_ptr_wrapper< T >

A wrapper for implicit conversion between various byte pointer types.

Ideally, std::byte would be used whenever representing raw bytes. However, often for legacy and ABI reasons, most interfaces use other C byte types such as char*, unsigned char*, and void*. This wrapper provides implicit conversion to and from these byte pointer types. When an implicit conversion is required, if the underlying pointer is statically convertible to the new type then a constexpr-specified static_cast implicit conversion operator will be used. Otherwise, the non-constexpr-specified reinterpret_cast implicit conversion operator will be used.

If the underlying pointer type is not void*, the wrapper satisfies the contiguous_iterator concept.

Note
This isn't a smart-pointer. The pointer should be non-owning!
