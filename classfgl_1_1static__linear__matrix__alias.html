Detailed Description

template<std::contiguous_iterator T_iter, major T_major, std::size_t ... T_bounds>
class fgl::static_linear_matrix_alias< T_iter, T_major, T_bounds >

A linear matrix alias which prioritizes static optimizations.

Remarks
Having static major, dimensions, and bounds makes certain compile-time optimizations possible and more likely to be used. For example: loop unrolling, constexpr calculation of offsets, reduced data duplication, and shared static_linear_matrix_info memory footprint between static aliases of the same major and bounds.
Template Parameters
T_iterThe underlying linear contiguous iterator type
T_majorThe major order of the matrix
T_boundsThe multi-dimensional bounds of the matrix
